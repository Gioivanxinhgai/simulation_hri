Lưu Đồ & Trình Mô Phỏng Vi Điều Khiển 8051

Chương trình đếm số lần ngắt ngoài INT0 và hiển thị LED

Điều khiển mô phỏng
Tốc độ: 600ms
Trạng thái thanh ghi CPU
Thanh ghi R5 0
Cho phép ngắt EA 0
Ngắt ngoài EX0 0
Cạnh ngắt IT0 0
Hiển thị LED (DISPLAY_LED)
000
Hoạt động hệ thống (Logs)
[00:00]Hệ thống đã sẵn sàng. Hãy bấm "Từng bước" hoặc "Tự động".
🔷 Chương Trình Chính (MAIN)
⚡ Ngắt Ngoài INT0 (ISR_INT0)

📄 Mã nguồn Assembly 8051

    ORG 0000H
    LJMP MAIN
    ORG 0003H
    LJMP ISR_INT0
MAIN:
    MOV R5, #0             ; Khởi tạo bộ đếm
    SETB IT0              ; Ngắt cạnh xuống
    SETB EX0              ; Cho phép ngắt INT0
    SETB EA               ; Cho phép ngắt toàn cục
Loop:
    LCALL DISPLAY_LED     ; Hiển thị LED
    SJMP Loop             ; Lặp vô hạn
ISR_INT0:
    INC R5               ; Tăng bộ đếm
    CJNE R5, #100, Exit ; Nếu R5 ≠ 100, thoát
    MOV R5, #0           ; Reset bộ đếm
Exit:
    RETI                  ; Trở về từ ngắt
